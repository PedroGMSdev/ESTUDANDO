# Curso Santander DIO

## Lógica de Programação em JavaScript

Este repositório foi criado para armazenar os exercícios, desafios e anotações desenvolvidos durante o curso **Lógica de Programação em JavaScript**, oferecido pela **DIO (Digital Innovation One)** em parceria com o **Santander**.

## Sobre o curso
O curso tem como objetivo desenvolver a base da lógica de programação utilizando **JavaScript**, abordando conceitos fundamentais que são essenciais para quem está iniciando na área de desenvolvimento de software.

Durante o curso, são trabalhados temas como:
- Conceitos básicos de lógica de programação
- Variáveis e tipos de dados
- Operadores
- Estruturas condicionais
- Estruturas de repetição
- Funções
- Classes e Objetos
- Resolução de problemas com JavaScript

## Tecnologias utilizadas
- **JavaScript**
- **Node.js** (quando necessário para executar os scripts)

## Objetivo
Este repositório tem como objetivo:
- Praticar os conceitos aprendidos no curso
- Acompanhar minha evolução em lógica de programação
- Servir como material de consulta futura
- Demonstrar aprendizado em JavaScript para oportunidades na área de tecnologia
