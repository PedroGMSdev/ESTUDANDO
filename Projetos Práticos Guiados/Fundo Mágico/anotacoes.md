Um projeto que usa IA para gerar cores para fundos, com N8N.

EXTENSÃO VS CODE ACONSELHADA:
- Portuguese (Brazil) Lenguage
- Dracula Theme
- vscode-icons
- Color Highlight

**HTML**

1 - uma TAG pode ter duas classes, basta separar as classes por espaço. Exp.: classe "card" e classe "main-card".
2 -  o **placeholder** serve para acrescentar um texto dentro do input, que serve como explicação, exemplo ou dica de uso. O **rows** vai dizer qual o tamanho inicial da altura de um textarea.

DICA: no HTML, assim como o rows, você pode usar o **cols** para regular a largura.
    Exp.:
        rows="5" cols="30"

3 - o **pre** serve para mostrar exatamente como você digitou no HTML.
